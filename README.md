# BeatSaber-Installer
Download your BeatSaber Levels by ID instead of downloading the zip file and unzipping the file from BeatSaver.com

# Commands
```
python main.py --id [beatsaber_song_id]
```
```
python main.py --multiple [search_setting]
 - 1 Trending",
 - 2 Date Ranked",
 - 3 Scores Set",
 - 4 Star Rating",
 - 5 Author"
```
